# 📄 OCR Document Scanner

An end-to-end AI system that extracts text from images and converts it into structured data using OCR and NLP.

---

# Features

Upload images (PNG, JPG, JPEG)

Extract text using OCR (Tesseract)

Image preprocessing with OpenCV

Structured data extraction (NLP + Regex)

FastAPI backend

Streamlit frontend

JSON API response

---

# Extracted fields:

Organizations

Dates

Emails

Phone numbers

Amounts

---

# Tech Stack

FastAPI

Streamlit

Tesseract OCR

OpenCV

spaCy
